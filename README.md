# cpp-cf-url

A utility for `CFURL`.

## Requirements

cpp-cf-url depends the following classes.

- [pqrs::cf::cf_string](https://github.com/pqrs-org/cpp-cf-string).

## Install

Copy `include/pqrs` and `vendor/vendor/include` directories into your include directory.
